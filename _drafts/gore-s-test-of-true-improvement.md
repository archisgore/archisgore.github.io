When done correctly… Assembly Language is beautiful. When done correctly, you don’t need privileged modes in processors — all programs can run in kernel space. When done correctly, cooperative multitasking is all you need. When used correctly, C has no memory-safety problems.

The problem is that the prefix “When done correctly”, makes every single statement a Tautology (makes them philosophically and literally true.)
