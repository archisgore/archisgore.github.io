Archis Gore

Engineer, founder, security researcher.

I work on diversity as a defense — the idea that the best way to defeat an attacker writing one exploit is to make every target different. The work spans the stack: lexer tokens, bytecode, calling conventions, kernel ABI, custom ISA. The same closure-based thesis, re-instantiated at every layer.

Current

tone.wine / WineTone

A wine recommender that calibrates to your vocabulary. Demonstrates a thesis about recommendation systems: words have context, but users have personal context on how they use words.

Live · 164K wines · Apache-2.0

encrypted-linux

A Linux distribution where every build is its own dialect. Scrambling-GCC produces a unique binary ABI per seed, so the same exploit can't move between machines.

In progress · GCC + dpkg + qemu

Foundational work

Encrypted Execution

Whitepaper that frames closure-based diversity as a defense primitive. The patent (USPTO 10,733,303) is pledged to the public domain.

2018–present · encrypted-execution.com

Polyverse · Polymorphic Linux

Founding CTO. Moving-target-defense product that recompiled every Linux binary with a different gadget layout, breaking exploit reusability industry-wide.

Acquired · 2015–2021

Polyscripting · PHP / WordPress

Reference implementation of the polyscripting idea at the interpreter level: per-build lexer dialects so an attacker's injected PHP doesn't even tokenise.

Reference impl · Apache-2.0

Open source
